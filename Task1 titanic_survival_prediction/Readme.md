
Task 1: Titanic Survival Prediction
The goal of this project is to build a Data Sceince model that predicts whether a passenger survived the Titanic disaster based on passenger details.

Dataset
The dataset contains passenger information such as:

Passenger class.
Age.
Gender.
Fare.
Number of family members onboard.
File used:

train.csv
titanic.py
Approach
Loaded and explored the dataset
Cleaned missing and inconsistent data
Converted categorical values into numerical format
Split data into training and testing sets
Trained a data science    classification model
Evaluated model    performance
Technologies Used
Python
Pandas
NumPy
Scikit-learn
How to Run the Project
python titanic.py
